Администрирование PostgreSQL
############################

АННОТАЦИЯ

Электронное учебно-методическое пособие «PostgreSQL» предназначено для обучающихся по профессиям и специальностям направления подготовки 09.00.00 Информатика и вычислительная техника, в том числе при изучении дисциплины «Управление и автоматизация баз данных».
Пособие включает теоретический материал, задания для самостоятельного выполнения, методические указания и примеры выполнения заданий. Учебный материал в пособии представлен по принципу «от простого к сложному»; задания, предлагаемые к выполнению, содержат иллюстрации, гиперссылки на источники и пошаговые практические рекомендации.

Для освоения дисциплины обучающемуся предлагается, используя удобную систему навигации, последовательно переходить от темы к теме, изучать теоретический материал, разбирать примеры и выполнять практические задания.

Электронное учебно-методическое пособие «Администрирование PostgreSQL» может быть использовано преподавателем для подготовки и проведения аудиторных учебных 
занятий, для организации самостоятельной работы обучающихся, а также при реализации дополнительных профессиональных программ для различных категорий слушателей.
